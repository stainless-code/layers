## Seam notes

- Don't use RAC's trigger wrappers — controlled overlay while mounted.
- **Exit-animation crux:** if `call.end` fires the moment the user dismisses, Layers unmounts synchronously and the exit animation never plays. RAC fix: hold a local `exiting` state, pass `isExiting`, and call `call.end` on `onAnimationComplete` (or RAC's `data-[exiting]:` classes).
- Lib owns focus/portal/scroll-lock — don't double-apply Layers-side equivalents.
