## Seam notes

- Don't use RAC's trigger wrappers — controlled overlay while mounted.
- **Exit-animation crux:** if `call.end` fires the moment the user dismisses, Layers unmounts synchronously and the exit animation never plays. Hold local `exiting` state and defer `call.end` until the CSS exit finishes (timer, `onAnimationComplete`, or RAC `data-[exiting]` — recipe uses a timer).
- Lib owns focus/portal/scroll-lock — don't double-apply Layers-side equivalents.
