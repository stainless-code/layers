## Seam notes

- Don't use RAC `DialogTrigger` — Layers owns the imperative open; the layer component renders the controlled overlay directly while mounted.
- Nest via the Layers **layer group** (`useLayerGroup` + `<group.Outlet />`), NOT stacked-in-one-component — one RAC `Modal` per layer.
- RAC owns portal/focus/scroll-lock per dialog — don't double-apply Layers-side equivalents.
- Don't drive RAC exit timing via Layers' `transition` axis — defer `call.end` until the lib animation finishes.
