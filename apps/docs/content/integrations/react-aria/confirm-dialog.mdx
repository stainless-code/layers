## Seam notes

- Don't use RAC's `DialogTrigger` — render the controlled overlay directly (`isOpen` literal `true`; `onOpenChange(false)` is the dismiss signal).
- Blockers: for Layers blockers that must prevent dismissal, use RAC's `isKeyboardDismissDisabled` + `isDismissable={false}` — don't also apply Layers-side focus/scroll-lock (RAC owns portal/focus/scroll-lock).
- Exit animation: instant-dismiss can `call.end` on `onOpenChange`; animated exit should hold local `exiting` + `onAnimationComplete` → `call.end` (see animated-enter-exit). Don't drive RAC exit timing via Layers' `transition` axis.
