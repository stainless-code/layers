## Seam notes

- Don't use RAC's trigger wrappers — Layers owns the imperative open; the layer component renders the controlled overlay directly while mounted (`isOpen` literal `true`, `onOpenChange(false)` is the dismiss signal).
- `call.end(result)` returns the typed boolean to the caller — same contract as the inline recipe.
- RAC owns focus trap / portal / scroll-lock — don't double-apply Layers-side equivalents.
- For animated slide-over exit, hold local `exiting` and `call.end(result)` on `onAnimationComplete`. Don't drive RAC exit timing via Layers' `transition` axis.
