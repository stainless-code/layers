## Seam notes

- Don't use Base UI's trigger wrappers — Layers owns the imperative open; the layer component renders the controlled overlay directly while mounted.
- **Exit-animation crux:** don't `call.end` on dismiss intent or Layers unmounts before the exit anim plays. Base UI fix: `setOpen(false)` on dismiss intent, resolve `call.end(result)` in `onOpenChangeComplete(false)` (fires AFTER Base UI's exit animation). Use `data-[starting-style]`/`data-[ending-style]` for the transitions.
- Lib owns focus/portal/scroll-lock — don't double-apply Layers-side equivalents.
