## Seam notes

- Don't use Base UI's trigger wrappers — controlled overlay while mounted.
- **Exit-animation crux:** don't `call.end` on dismiss intent or Layers unmounts before the exit anim plays. Base UI fix: `setOpen(false)` on dismiss intent, resolve `call.end(result)` in `onOpenChangeComplete(false)` (fires after Base UI's exit animation). Style with Base UI's starting/ending data attributes as needed.
- Lib owns focus/portal/scroll-lock — don't double-apply Layers-side equivalents.
