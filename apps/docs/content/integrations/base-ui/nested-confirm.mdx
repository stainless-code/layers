## Seam notes

- Don't use `Dialog.Trigger` — one controlled `Dialog.Root` per layer.
- Nest via the Layers layer group (`useLayerGroup` + `<group.Outlet />`), one `Dialog.Root` per layer — not stacked-in-one-component.
- **Exit-animation crux** — `onOpenChangeComplete(false)` → `call.end` per dialog; Base UI owns portal/focus/scroll-lock.
