## Seam notes

- Don't use Base UI's `Dialog.Trigger` — Layers owns the imperative open; controlled `open` starts `true`.
- **Exit-animation crux (Base UI):** on dismiss intent set pending result + `setOpen(false)`; resolve `call.end(result)` in `onOpenChangeComplete(false)` (fires AFTER Base UI's exit animation) so exit anim is never cut off.
- Blockers: use Base UI's dismiss-disable (`disablePointerDismissal` / `eventDetails.cancel()`).
- Lib owns portal/focus/scroll-lock — don't double-apply Layers-side equivalents.
