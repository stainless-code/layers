A delete button in a sidebar needs a yes/no. `useLayer(confirm)` + `await confirm.open({ title })` returns the typed boolean.

:::note[Live]
This demo runs the real `@stainless-code/react-layers` in your browser (React).

live under [Integrations](/integrations).
:::

## Layers wiring

Declare a typed layer, bind it with `useLayer(confirm)` (or `createLayer` headless), then `await confirm.open(payload)` for a typed boolean. The recipes below wrap this contract in a real dialog.
