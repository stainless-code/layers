Dirty form — Close shouldn't nuke unsaved edits. `call.addBlocker` vetoes dismiss; a discard confirm via `useLayerGroup` asks first; `{ force: true }` bypasses when the user confirms.

:::note[Live]
This demo runs the real `@stainless-code/react-layers` in your browser (React).

:::

## Layers wiring

Register a blocker that returns `false` to veto dismissal; `force: true` overrides it. The discard confirm is a child layer opened via `useLayerGroup`. The recipes below wrap it in real dialogs.
