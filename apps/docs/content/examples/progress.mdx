A file upload needs a live percent bar without remounting. `useLayer(progress)` opens once; `progressLayer.stack.update(layer, { percent })` pushes ticks while it stays mounted.

:::note[Live]
This demo runs the real `@stainless-code/react-layers` in your browser (React).
Other frameworks are code-complete below.
:::
