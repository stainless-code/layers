## Variations

- **Hook-based open** — `useLayerClient()` inside a component; see [Confirm dialog](/examples/confirm-dialog)
- **Payload validation** — reject bad input before mount; see [Payload validation](/guides/payload-validation)
- **Shared client** — pass one `LayerClient` across routes via `StackProvider`; see [Getting started](/guides/getting-started)
