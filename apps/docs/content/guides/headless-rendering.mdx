`StackOutlet` maps a stack snapshot to registered components — the convenience path. When you need custom portals, per-layer switches, or non-component renderers, use the headless primitives.

## useStackHandles

Returns `{ states, getCall }` — everything `StackOutlet` uses internally:

<Tabs>
<Tab title="React" icon="/icons/react.svg">

```tsx lineNumbers
import { useStackHandles } from "@stainless-code/react-layers";

function CustomHost() {
  const { states, getCall } = useStackHandles("modal");
  return (
    <Portal>
      {states.map((s) => {
        const call = getCall(s);
        return (
          <MyModal
            key={s.id}
            call={call}
            payload={s.payload}
            transition={s.transition}
          />
        );
      })}
    </Portal>
  );
}
```

import { For } from "solid-js";
import { useStackHandles } from "@stainless-code/solid-layers";

function CustomHost() {
  const { states, getCall } = useStackHandles("modal");
  return (
    <Portal>
      <For each={states()}>
        {(s) => (
          <MyModal
            call={getCall(s)}
            payload={s.payload}
            transition={s.transition}
          />
        )}
      </For>
    </Portal>
  );
}

```ts lineNumbers
import { Component } from "@angular/core";
import { useStackHandles } from "@stainless-code/angular-layers";

@Component({
  template: `
    @for (s of states(); track s.id) {
      <my-modal
        [call]="getCall(s)"
        [payload]="s.payload"
        [transition]="s.transition"
      />
    }
  `,
})
class CustomHost {
  private handles = useStackHandles("modal");
  states = this.handles.states;
  getCall = this.handles.getCall;
}
```

<Tab title="Svelte (runes)" icon="/icons/svelte.svg">

```svelte lineNumbers
<script lang="ts">
  import { useStack } from "@stainless-code/svelte-layers";

  const stack = useStack({ stack: "modal" });
</script>

{#each stack.current as s (s.id)}
  {@const call = stack.callFor(s)}
  {#if call}<MyModal {call} payload={s.payload} transition={s.transition} />{/if}
{/each}
```

`useStack().current` + `callFor` — no separate `useStackHandles` export.

</Tab>
<Tab title="Svelte (store)" icon="/icons/svelte.svg">

```svelte lineNumbers
<script lang="ts">
  import { callFor, setLayerClient, useLayerClient, useStack } from "@stainless-code/svelte-layers/store";

  setLayerClient();
  const client = useLayerClient();
  const stack = useStack({ stack: "modal" });
</script>

{#each $stack as s (s.id)}
  {@const call = callFor(client, "modal", s)}
  {#if call}<MyModal {call} payload={s.payload} transition={s.transition} />{/if}
{/each}
```

Store entry subscribes via `$stack` + standalone `callFor(client, stackId, state)`.

`StackOutlet` deliberately has **no** `render` prop — headless is the escape hatch.

## StackSubscribe

Isolate a selector in JSX/template without pulling the full snapshot into your component:

<Tabs>
<Tab title="React" icon="/icons/react.svg">

```tsx lineNumbers
import { StackSubscribe } from "@stainless-code/react-layers";

function OpenCount() {
  return (
    <StackSubscribe stack="modal" selector={(s) => s.length}>
      {" "}
      {(count) => <span>{count} open</span>}
    </StackSubscribe>
  );
}
```

import { Component } from "@angular/core";
import { useStack } from "@stainless-code/angular-layers";

@Component({ template: `<span>{{ count() }} open</span>` })
class OpenCount {
  count = useStack({ stack: "modal", select: (s) => s.length });
}

No `StackSubscribe` — `useStack({ stack, select })` as a signal is the equivalent.

For fully-typed selector output in Vue, prefer `useStack({ stack, select })` in `setup()`.

## Adapter equivalents

| Adapter        | Headless path                                | Built-in outlet           |
| -------------- | -------------------------------------------- | ------------------------- |
| React / Preact | `useStackHandles`, `StackSubscribe`          | `StackOutlet`             |
| Solid / Vue    | `useStackHandles`, `StackSubscribe`          | `StackOutlet`             |
| Angular        | `useStackHandles`, `renderStack(vcr)`        | `renderStack`             |
| Svelte (runes) | `useStack().current` + `callFor`             | — (render in your markup) |
| Svelte (store) | `$stack` + `callFor(client, stackId, state)` | —                         |

:::tip
Svelte's `useStack()` already returns the headless pair — `.current` for states and `callFor` for call contexts. No separate `useStackHandles` export.
:::

Angular has no `StackSubscribe`; `useStack({ stack, select })` as a signal is the idiomatic equivalent.

## When to skip StackOutlet

- Custom portal targets or z-index layering
- Switching on `state.key` to pick arbitrary components (not the registered `component`)
- Integrating with a design-system modal primitive that owns its own DOM
- Angular and Svelte — imperative/primitive rendering by design

Dev warning: `StackOutlet` logs when a layer has no `component` and renders nothing.
