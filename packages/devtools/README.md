# @stainless-code/layers-devtools

Solid panel + EventClient bridge for Layers in [TanStack Devtools](https://tanstack.com/devtools/latest).

> Experimental — the API may change between minor releases. Pin your version.

## Install

Usually install the React doorbell instead — it depends on this package:

`bun add -d @stainless-code/react-layers-devtools @tanstack/react-devtools`
